# Hero covers · builder de Mi Tarjeta Pro

Fotos de cover para el hero (`heroBackgroundUrl`), una por categoría de negocio.
`js/mi-tarjeta.js` (`HERO_COVER_BY_CATEGORY`) las asigna según la categoría elegida
en el formulario del builder, y `negocio/negocio.js` + `negocio/negocio.css` las
renderizan en `.biz-hero--photo` con overlay oscuro — el mismo motor que usan las
tarjetas ya publicadas.

Archivos (JPG, 1600×900 px, <200 KB):

- `comida.jpg`
- `belleza.jpg`
- `profesional.jpg`
- `universal.jpg`
